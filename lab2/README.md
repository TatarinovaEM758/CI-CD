# 	Цель работы:

Научиться разрабатывать воспроизводимые аналитические инструменты. Пройти полный цикл: от написания Python-скрипта для обработки бизнес-данных до его упаковки в Docker-образ и запуска в изолированной среде.

## Ход выполнения работы

### 1. Создание структуры проекта

```bash
mkdir real-estate-cleaner
cd real-estate-cleaner
mkdir app
```

1. Генерация "грязных" данных о недвижимости с использованием библиотеки Faker:

 - Поля: район, площадь, цена, этаж, год постройки, тип материала

 - 10-15% пропусков (null-значений) в каждом поле

![](https://github.com/TatarinovaEM758/CI-CD/blob/main/lab2/img/1кон.png)

2. Очистка данных:

 - Числовые пропуски заполняются медианным значением

 - Категориальные пропуски заполняются модальным значением или значением "не указан"

3. Аналитика:

 - Расчет цены за квадратный метр

 - Группировка по районам (средняя, медианная, мин., макс. цена за м²)

 - Общая статистика по всем объектам

 - Анализ по типу материала

4. Сохранение результата:

 - Отчет сохраняется в файл cleaned_data.txt

 - Содержимое файла выводится в консоль
